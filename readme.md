# Program memindahkan cairan dari satu gelas ke gelas yang lain

#### Progaram ini menggunakan JS DOM, untuk manipulasi perpindahan cairan dengan eventListener click dimana jika salah satu gelas yang terisi dengan liquid diklik maka akan disimpan isinnya, lalu jika gelas yang kosong diklik maka gelas yang kosong tersebut akan diisi oleh cairan dari gelas yang diklik diawal.
