# Program memindahkan cairan dari satu gelas ke gelas yang lain

#### Progaram ini menggunakan JS DOM, untuk manipulasi perpindahan cairan dengan eventListener click dimana jika salah satu gelas yang terisi dengan liquid diklik maka akan dicari salah satu gelas yang kosong lalu gelas tersebut diisi oleh cairan dari gelas yang diklik.
